## 快速入门

### 第一次登录

1. 打开浏览器访问 WPForge 管理后台
2. 使用管理员账号登录
3. 首次登录后，建议先修改默认密码

### 系统概览

登录后进入仪表盘，您可以看到：
- **统计卡片**: 站点数量、产品数量、任务数量等关键指标
- **最近任务**: 最近执行的任务列表和状态
- **系统状态**: 系统运行状态和资源使用情况
- **快速操作**: 常用功能的快捷入口

### 基本工作流程

1. **添加站点**: 连接您的 WordPress 网站
2. **采集产品**: 从目标网站采集产品数据
3. **翻译处理**: 自动翻译产品内容
4. **SEO优化**: 优化产品SEO信息
5. **导入WordPress**: 将产品导入到您的网站

## 站点管理

### 添加站点

1. 进入「站点管理」页面
2. 点击「添加站点」按钮
3. 填写站点信息：
   - **站点名称**: 便于识别的名称
   - **WordPress地址**: 您的网站URL
   - **用户名**: WordPress管理员用户名
   - **应用密码**: WordPress应用密码（推荐）或密码
   - **WooCommerce API密钥**: 可选，用于WooCommerce操作
4. 点击「测试连接」验证连接是否正常
5. 点击「保存」完成添加

### 站点设置

每个站点可以单独配置：
- **语言设置**: 站点默认语言
- **货币设置**: 站点默认货币
- **页面构建器**: Elementor / Bricks / Gutenberg
- **SEO设置**: SEO相关配置
- **导入设置**: 产品导入相关配置

### 站点分组

您可以将站点按业务、地区等进行分组，便于批量管理。

## 产品采集

### 快速采集

1. 进入「产品采集」页面
2. 在「快速采集」标签页中
3. 输入目标产品URL
4. 点击「开始采集」
5. 系统会自动识别并采集产品信息

### 批量采集

1. 进入「产品采集」页面
2. 在「批量采集」标签页中
3. 输入目标网站URL（产品列表页或分类页）
4. 配置采集参数：
   - **最大采集数量**: 最多采集多少个产品
   - **采集图片**: 是否采集产品图片
   - **采集变体**: 是否采集产品变体
   - **采集评论**: 是否采集产品评论
   - **使用代理**: 是否使用代理IP
5. 点击「开始采集」
6. 在任务列表中查看采集进度

### AI智能识别

WPForge 使用AI自动识别目标网站：
- 自动识别网站平台类型（Shopify、WooCommerce等）
- 自动识别产品字段结构
- 自动识别分页方式
- 自动识别网站语言和货币
- 自动检测反爬机制

### 采集历史

在「采集历史」页面可以查看所有采集任务：
- 任务状态：待处理、进行中、已完成、失败、已取消
- 采集统计：总数量、成功、失败、跳过
- 任务详情：查看具体采集了哪些产品
- 重新采集：对失败的任务重新执行

## 翻译管理

### 文本翻译

1. 进入「翻译中心」页面
2. 在「文本翻译」标签页中
3. 输入要翻译的文本
4. 选择目标语言
5. 选择翻译引擎（可选，默认自动选择）
6. 点击「翻译」

### 批量翻译

1. 进入「翻译中心」页面
2. 在「批量翻译」标签页中
3. 选择要翻译的产品
4. 选择目标语言
5. 选择要翻译的字段（标题、描述、短描述等）
6. 选择翻译引擎
7. 点击「开始翻译」

### 翻译引擎

WPForge 支持多种翻译引擎：
- **DeepL**: 翻译质量高，支持多种语言
- **Google翻译**: 支持语言最多
- **OpenAI**: AI翻译，可自定义风格
- **Anthropic**: AI翻译，质量优秀
- **本地模型**: 私有化部署，数据安全

系统会自动选择最合适的翻译引擎，并支持引擎自动降级。

### 术语库

您可以创建和管理翻译术语库，确保专业术语翻译一致：
- 添加术语：源语言术语 + 目标语言翻译
- 术语分类：按行业、产品等分类
- 导入导出：批量导入导出术语

### 翻译记忆

系统会自动记录翻译历史，相同内容会复用之前的翻译结果，提高效率和一致性。

## SEO优化

### SEO审计

1. 进入「SEO优化」页面
2. 在「SEO审计」标签页中
3. 输入要审计的URL或选择产品
4. 点击「开始审计」
5. 查看审计结果：
   - **总体评分**: 综合SEO评分
   - **内容评分**: 内容质量评分
   - **技术评分**: 技术SEO评分
   - **性能评分**: 页面性能评分
   - **问题列表**: 发现的SEO问题
   - **改进建议**: 具体的优化建议

### 批量SEO优化

1. 进入「SEO优化」页面
2. 在「批量优化」标签页中
3. 选择要优化的产品或文章
4. 选择优化项目：
   - 优化标题
   - 优化描述
   - 优化关键词
   - 生成Schema
5. 点击「开始优化」

### Schema结构化数据

WPForge 支持多种Schema类型：
- **产品 (Product)**: 产品信息、价格、评分等
- **文章 (Article)**: 文章标题、作者、发布时间等
- **面包屑 (Breadcrumb)**: 页面导航路径
- **评价 (Review)**: 用户评价和评分
- **组织 (Organization)**: 企业组织信息
- **FAQ**: 常见问题解答
- **HowTo**: 操作指南

### XML Sitemap

系统会自动生成XML Sitemap，帮助搜索引擎更好地索引您的网站。

## WordPress导入

### 导入方式

WPForge 支持三种导入方式，系统会自动选择最优方式：

1. **REST API** (推荐): 速度最快，需要WooCommerce API权限
2. **WP-CLI**: 次选方式，需要服务器SSH权限
3. **浏览器模拟**: 兜底方案，无需特殊权限

### 导入产品

1. 进入「WordPress导入」页面
2. 选择目标站点
3. 选择要导入的产品（可全选或按分类选择）
4. 配置导入选项：
   - **更新已存在产品**: 产品已存在时是否更新
   - **导入图片**: 是否导入产品图片
   - **导入变体**: 是否导入产品变体
   - **自动发布**: 导入后自动发布产品
   - **图片压缩**: 自动压缩图片
   - **生成WebP**: 生成WebP格式图片
   - **自动ALT**: 自动生成图片ALT标签
5. 点击「开始导入」
6. 查看导入进度和结果

### 图片智能处理

导入时自动处理图片：
- **图片压缩**: 自动压缩图片大小
- **WebP转换**: 生成WebP格式图片
- **ALT标签**: 自动生成图片ALT文本
- **图片去重**: 自动检测并跳过重复图片
- **尺寸调整**: 自动调整图片尺寸

### 分类自动创建

系统会自动创建产品分类，保持与原网站一致的分类结构。

## 代理管理

### 代理池

WPForge 支持动态代理系统：
- **多供应商支持**: BrightData、Oxylabs、Smartproxy等
- **自动质量检测**: 自动检测代理质量和速度
- **GEO地理位置匹配**: 根据目标网站自动选择对应地区的代理
- **自动轮换**: 自动轮换代理IP，避免被封禁

### 添加代理

1. 进入「代理管理」页面
2. 点击「添加代理」
3. 填写代理信息：
   - **代理地址**: 代理服务器地址
   - **端口**: 代理端口
   - **协议**: HTTP / HTTPS / SOCKS5
   - **用户名**: 代理用户名（可选）
   - **密码**: 代理密码（可选）
   - **国家/地区**: 代理所在地区
   - **服务商**: 代理服务商
4. 点击「保存」

### 批量导入

您可以批量导入代理列表，支持TXT、CSV等格式。

### 代理检测

系统会定期检测代理可用性和响应速度，自动剔除不可用的代理。

### 代理策略

可以配置代理选择策略：
- **轮询**: 按顺序轮流使用代理
- **随机**: 随机选择代理
- **最优**: 选择响应速度最快的代理
- **按地区**: 根据目标地区选择对应代理

## 常见问题

### 1. 采集失败怎么办？

- 检查目标网站是否可以正常访问
- 尝试使用代理IP
- 降低采集速度
- 检查是否有验证码需要处理

### 2. 翻译不准确怎么办？

- 尝试使用不同的翻译引擎
- 添加术语库条目
- 使用AI翻译并调整提示词

### 3. 导入WordPress失败怎么办？

- 检查WordPress连接是否正常
- 检查API密钥权限是否足够
- 检查PHP内存限制是否足够
- 尝试减少单次导入数量

### 4. 如何提高采集速度？

- 增加并发数（注意不要触发反爬）
- 使用高质量代理
- 跳过图片采集（后续单独处理）
- 使用增量采集模式

### 5. 数据安全吗？

- 所有数据都存储在您自己的服务器上
- 支持私有化部署
- API密钥等敏感信息加密存储
- 定期自动备份
