## 项目概述

WPForge是一款全智能化的WordPress建站与自动化平台，专为跨境电商和SEO优化设计。通过AI技术实现一键采集、自动翻译、智能定价、批量建站等功能。

**技术栈**:
- 后端: Python 3.11+ + FastAPI + SQLAlchemy + Celery
- 前端: Vue 3 + Pinia + TypeScript + Element Plus
- 数据库: PostgreSQL 15 + Redis 7
- 浏览器自动化: Playwright + 完整反检测体系
- WordPress生态: WPForge Theme + Elementor + Bricks + WooCommerce
- 部署: Docker + Docker Compose + Nginx
